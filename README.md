# Guess the Number Game

## Description
**Guess the Number** is a single-player interactive game where the player is prompted to guess a randomly generated number between 1 and 100. 

The game provides continuous feedback, letting the player know if their guess is too high or too low, until they successfully guess the correct number.

## Objective
The objective of the game is to guess the randomly generated number between 1 and 100.

## Features
- Random number generation between 1 and 100.
- Interactive user input for guessing.
- Real-time feedback on guesses ("Too high" or "Too low").
- Success message upon guessing the correct number.

## How to Play
1. Run the Python script or Jupyter Notebook (`checkpoint.ipynb`).
2. The game will prompt you to enter a guess.
3. Type a number between 1 and 100 and press Enter.
4. Read the feedback provided by the game:
   - If it says **"Too high"**, try a lower number.
   - If it says **"Too low"**, try a higher number.
5. Keep guessing until you find the correct number!

## Requirements
- Python 3.x (or Jupyter Notebook environment to run the `.ipynb` file)
